# Jogo RPG
Um jogo de RPG em python, atualmente jogavel apenas pelo terminal, com foco em:

- Sistema de combate
- Progressão de nivel
- Loja
- Inventario

Projeto atualmente em desenvolvimento porem ira ser escalado para o **pygame**

## Intuito do projeto

- Aperfeiçoar conhecimentos na linguagem python

- Aprender e praticar conceitos da OOP(Programação orientada a objetos) com uso de classes

## Como jogar

1- Clone o repositorio
```bash
git clone https://github.com/Miguel00Roza/RpgGame
cd RpgGame
```

2- Crie e ative o ambiente virtual (opcional)

```bash
python -m venv .venv
source .venv/bin/activate # Linux/Mac
./.venv/Scripts/activate # Windows
```

3- Instale as dependencias

```bash
pip install -r requirements.txt
python src/main.py 
```

4- Rode o jogo

```bash
python src/main.py
```
